Application description: 
- must be launched with a command line argument (-h lists all available arguments)
- can be used to calculate area and capacity of given dimensions (these functions are defined in DLL)
- keeps track of run count using registry
- uses different registry key depending on if running debug or release version (conditional constant compilation)
- defines static literals in their own Constants-class
- implements Doxygen documentation (in separate directory)
